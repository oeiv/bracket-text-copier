Frequently Asked Questions

Does it work on all websites?

Yes! The extension works on all websites by default. You can disable it for specific sites using the popup menu.

What types of brackets are supported?

Currently, the extension supports square brackets [ ]. Support for other bracket types may be added in future versions.

Is my data safe?

Absolutely! The extension doesn't collect, store, or transmit any of your data. Everything happens locally in your browser.

Can I contribute to the project?

Yes! This is an open-source project. Visit our GitHub repository to contribute.
